## Mechanism

- The SVG sets `width="100%"` with a fixed `viewBox="0 0 370 57"`, so it stretches to fill the parent's width while preserving aspect ratio.
- Every path uses `fill="currentColor"` — the wordmark inherits its color from the nearest ancestor with a `color` value (i.e. any Tailwind `text-*` utility).
- `aria-label="Miscreants"` and `role="img"` make it announce as a single image to assistive tech.

## Usage

```astro
---
import Logo from "@components/Logo.astro";
---

<Logo class="h-5 w-auto" />

<div class="text-canvas">
  <Logo class="h-8 w-auto" />
</div>
```

## Notes

- Use `h-*` together with `w-auto` to size the logo. Without `w-auto`, the SVG's `width="100%"` will fill the parent's width.
- The registered-trademark mark is part of the path data — not a separate text run.
